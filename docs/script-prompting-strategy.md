# 스크립트 프롬프트 전략

Claude Script Generation Engine은 프롬프트를 한 덩어리로 보내지 않고, 다음 블록으로 분리해 조합한다.

## 1. 이슈 블록

- 이슈 제목
- 이슈 요약
- 현재 다루는 질문

## 2. 근거 블록

- 검증된 공식 통계 목록
- 보조 시장 맥락 목록
- 각 근거의 `evidence_id`, 출처, 발표일, 값, 메모

## 3. 연출 블록

- 스타일 프리셋
- 톤
- 대상 오디언스
- 사용자 추가 지시

## 4. 출력 제약 블록

- JSON만 반환
- 섹션/장면/근거 매핑 스키마 고정
- 숫자 주장은 반드시 `evidence_ids`에 연결
- 시장 데이터는 보조 자료로만 사용

## 5. 재생성 전략

- 섹션 재생성 시 대상 섹션과 나머지 섹션 문맥을 분리 제공
- 전체 스크립트 스타일은 유지하고, 해당 섹션만 국소적으로 수정
- 재생성 결과도 JSON 단일 섹션 구조로 고정

## 6. 저장 전략

- 생성 결과는 prompt snapshot 안에 제목, 요약, 아웃라인, 섹션, 장면, 근거 매핑을 함께 저장한다
- 섹션 재생성 시에도 최신 전체 스크립트 snapshot으로 다시 기록해 버전별 비교가 가능하도록 유지한다
